# 版本控制

## [1.0.0] - 2025-11-28

### ✨ 新增

- **核心功能**

  - 用戶註冊與登入系統（基於郵箱+密碼）
  - 密碼複雜度驗證（長度、大小寫、數字）
  - YOLOv11 實時病害檢測引擎
  - 檢測歷史記錄保存
  - 用戶隔離的檢測記錄管理

- **前端界面**

  - 現代化認證頁面（登入/註冊切換）
  - 響應式應用主界面
  - 圖像預覽功能
  - 檢測結果模態框
  - 歷史紀錄卡片展示
  - Bootstrap 5 美化 UI

- **API 端點**

  - `/register` - 用戶註冊
  - `/login` - 用戶登入
  - `/logout` - 用戶登出
  - `/check-auth` - 檢查認證狀態
  - `/predict` - 圖像預測
  - `/history` - 檢測歷史查詢

- **安全特性**
  - Werkzeug 密碼加密
  - 基於 Session 的認證
  - 用戶資料隔離
  - Base64 圖像傳輸

### 🔧 技術棧

- Flask 3.0+ 後端框架
- YOLOv11 物體檢測模型
- Bootstrap 5.3 前端框架
- JSON 資料存儲
- Gunicorn WSGI 伺服器

